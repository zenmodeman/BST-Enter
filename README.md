# BST-Enter
A simple game for entering base stats


## Run Locally

This is a static HTML/JS site, but it should be served over HTTP because the page loads `fully_evolved_base_stats.json` with `fetch()`.

Example with Python, from the project root, run:

```bash
python3 -m http.server 8000
```

Then open:

```text
http://localhost:8000/
```
